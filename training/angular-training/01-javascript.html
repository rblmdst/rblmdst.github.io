Types

Types : determine the operations that we can operate on a variable.

2 categories of Type System

Static Type vs Dynamic Type

Static Type

  • Variable type is determined at compile-time.
  • Variable type can not change during the program execution.

eg : C++, Java.

Dynamic Type

  • Variable type is determined at run-time.
  • Variable type can change during the program execution.

eg : Javascript, Python.

Javascript is dynamically typed language 


              // no error
              var message = "hello";
              message = 123456;

Control Structures

Conditional Structures allow to perform different actions based on conditions.

The If Statement

if...else 


              if (condition) {
                instructions1 // executed only when condition is evaluated to true
              } else {
                instructions2 // executed only when condition is evaluated to false
              }

if...else 


              var x = 15;
              if (x % 2 === 0) {
                console.log("x is a multiple of 2");
              } else {
                console.log("x is not a multiple of 2");
              }
              // Result
              "x is not a multiple of 2"

if..else if..else 


              if (condition1) {
                  instructions1 // executed only when condition1 is evaluated to true
              } else if (condition2) {
                  instructions2 // executed only when condition2 is evaluated to true
              } else {
                  instructions3 // executed only when condition1 nor condition2 is evaluated to true
              }

if..else if..else 


              var x = -12
              if (x > 0) {
                console.log("x is positive");
              } else if (x < 0) {
                console.log("x is negative");
              } else {
                console.log("x is equal to zero");
              }
              // Result
              "x is negative"

Loops

The For loop 


              for (initialExpression; condition; incrementExpression) {
                statement
              }

The For loop 


              var end = 5;
              for (var i = 0; i < end; i++) {
                if (i % 2 === 0) {
                  console.log(i);
                }
              }
              // Result
              0
              2
              4

The While loop 


              while (condition) {
                statement
              }

The While loop 


              var end = 5;
              var i = 0;
              while (i < end) {
                if (i % 2 === 0) {
                  console.log(i);
                }
                i++
              }
              // Result
              0
              2
              4

The switch statement

The Switch statement 


          switch (expression) {
            case value1:
              //Statements executed when the result of expression matches value1
              [break;]
            case value2:
              //Statements executed when the result of expression matches value2
              [break;]
            ...
            case valueN:
              //Statements executed when the result of expression matches valueN
              [break;]
            [default:
              //Statements executed when none of the values match the value of the expression
              [break;]]
          }

The Switch statement 


            var expr = 'Papayas';
            switch (expr) {
              case 'Oranges':
                console.log('Oranges are $0.59 a pound.');
                break;
              case 'Mangoes':
              case 'Papayas':
                console.log('Mangoes and papayas are $2.79 a pound.');
                break;
              default:
                console.log('Sorry, we are out of ' + expr + '.');
            }
            // Result
            "Mangoes and papayas are $2.79 a pound."

Functions

Functions allow creation of reusable block of codes.

Declaration


          function functionName(param1, param2, ..., paramN) {
            // function body
          }

          function square(x) {
            return x * x;
          };

Usage


          functionName(arg1, arg2, ..., argN);

          square(4); // 16

A function can have multiple parameters or no parameters at all.


            // function without parameters
            function greet() {
              console.log("Hello World!");
            };
            greet(); // "Hello World!"


            // function with parameters (a and b)
            function sum(a, b) {
              return a + b;
            };
            console.log(sum(3, 7)); // 10

A function can return a value or not.


            // function that does not returned a value
            function greet() {
              console.log("Hello World!");
            };
            greet(); // "Hello World!"


            // function that do returned a value
            function sum(a, b) {
              return a + b;
            };
            console.log(sum(3, 7)); // 10

Higher-order functions

Functions can be assigned to a variable as value 


              var sum = function (a, b) {
                return a + b;
              };
              console.log(sum(3, 7)); // 10

Functions can be pass as arguments to another function 


                function greet() {
                  console.log("Hello World!")
                };

                setTimeout(greet, 5000) // will print "Hello World" after 5 seconds

Functions can be returned as value by a function 


              function addX(x) {
                return function(y) {
                  return x + y;
                }
              };

              var addTen = addX(10);
              console.log(addTen(12)); // 22

Arrays

Declaration, size, access/update items by index


                var tab1 = []; // declare empty array
                var tab2 = [2, "test", true]; // declare array with values

                // get item by index
                console.log(tab1[0]); // undefined
                console.log(tab2[0]); // 2

                // change value of a specific index
                tab2[1] = false;
                console.log(tab2); // [2, false, true]

                // get size
                console.log(tab1.length); // 0
                console.log(tab2.length); // 3

Append item, remove last item


                var tab1 = []; // declare empty array
                var tab2 = [2, "test", true]; // declare array with values

                // append item
                tab1.push(15);
                console.log(tab1); // [15]
                tab1.push(8);
                console.log(tab1); // [15, 8]

                // remove last item
                var last = tab2.pop();
                console.log(last); // true
                console.log(tab2); // [2, "test"]
