Part 1 : Built-in Directives

Purpose

  • Know the difference between directives and components.
  • Learn Angular built-in directives.

Directives

Directives are components without a view. They are components without a template.

Directives : are entities that change the behavior of components or elements and are one of the core building blocks Angular.

There are two main types of directives :

  • Attribute directives : change the behavior of a component or element but don't affect the structure of the DOM.
  • Structural directives : change the the structure of the DOM.

Built-in attribute directives

NgStyle

[NgStyle] Directive

Allows to modify a component or element's style attribute. 


            @Component({
              selector: 'style-example',
              template: `
              <p style="padding: 1rem"
              [ngStyle]="{'font-weight': 'bold', borderBottom: borderStyle }">
              </p>
              `
            })
            export class StyleExampleComponent {
              borderStyle: string = '1px solid black';
            }

NgClass

[NgClass] Directive

Changes the class attribute that is bound to the component or element it's attached to.

There are a few different ways of using the directive :

  • Binding a string
  • Binding an array
  • Binding an object

[NgClass] (Binding a string)


              @Component({
                selector: 'class-as-string',
                template: `
                <p ngClass="centered-text underlined" class="orange"></p>
                `,
                styles: [`
                  .centered-text {
                    text-align: center;
                  }
                  .underlined {
                    border-bottom: 1px solid #ccc;
                  }
                  .orange {
                    color: orange;
                  }
                `]
              })
              export class ClassAsStringComponent {
              }

[NgClass] (Binding an array)


              @Component({
                selector: 'class-as-array',
                template: `
                <p [ngClass]="['warning', 'big']"></p>
                `,
                styles: [`
                  .warning {
                      color: red;
                      font-weight: bold;
                  }
                  .big {
                      font-size: 1.2rem;
                  }
                `]
            })
            export class ClassAsArrayComponent {
            }

[NgClass] (Binding an object)


              @Component({
                selector: 'class-as-object',
                template: `
                <p [ngClass]="{ card: true, dark: false, flat: flat }">
                  

                  
                </p>
                `,
                styles: [`
                  .card {
                    border: 1px solid #eee;
                    padding: 1rem;
                    box-shadow: 2px 2px 2px #888888;
                  }
                  .dark {
                    background-color: #444;
                    border-color: #000;
                    color: #fff;
                  }
                  .flat {
                    box-shadow: none;
                  }
                `]
            })
            export class ClassAsObjectComponent {
              flat: boolean = true;
            }

Built-in structural directives

NgIf

*ngIf Directive

Conditionally renders components or elements based on whether or not an expression is true or false. 


              @Component({
                selector: 'app',
                template: `
                  
                  <if-example *ngIf="exists">
                    Hello
                  </if-example>
                `
              })
              export class AppComponent {
                exists: boolean = true;
                toggleExists() {
                  this.exists = !this.exists;
                }
              }

*ngIf Directive


              <div *ngIf="hero" class="name">{{hero.name}} </div>

              

              <ng-template [ngIf]="hero">
                
{{hero.name}}

              </ng-template>

              
              <div *ngIf="counter > 1">example </div>

*ngIf..else 


              <div *ngIf="counter > 1; else elseBlock">example </div>
              <ng-template #elseBlock>else example </ng-template>

NgFor

*ngFor Directive

ngFor directive is a way of repeating a template by using each item of an iterable as that template's context.

*ngFor Directive


                @Component({
                  selector: 'app',
                  template: `
                    <for-example *ngFor="let episode of episodes" [episode]="episode">
                      {{episode.title}}
                    </for-example>
                  `
                })
                export class AppComponent {
                  episodes: any[] = [
                    { title: 'Winter Is Coming', director: 'Tim Van Patten' },
                    { title: 'The Kingsroad', director: 'Tim Van Patten' },
                    { title: 'The Pointy End', director: 'Daniel Minahan' }
                  ];
                }

*ngFor Directive

ngFor also provides some other values that can be bound to:

  • index - position of the current item in the iterable starting at 0
  • first - true if the current item is the first item in the iterable
  • last - true if the current item is the last item in the iterable
  • even - true if the current index is an even number
  • odd - true if the current index is an odd number

*ngFor Directive


            @Component({
              selector: 'app',
              template: `
              <for-example
                *ngFor="let episode of episodes; let i = index; let isOdd = odd"
                [episode]="episode"
                [ngClass]="{ odd: isOdd }">
                {{i+1}}. {{episode.title}}
              </for-example>
              `
            })
            export class AppComponent {
              // ...
            }

ngSwitch

[ngSwitch] Directives


              @Component({
                selector: 'app',
                template: `
                <div [ngSwitch]="door">
                  <door [id]="1" *ngSwitchCase="1">A new car!</door>
                  <door [id]="2" *ngSwitchCase="2">A washer and dryer!</door>
                  <door [id]="3" *ngSwitchCase="3">A trip to Tahiti!</door>
                  <door [id]="4" *ngSwitchCase="4">25 000 dollars!</door>
                  <door *ngSwitchDefault class="closed"></door>
                </div>
                

                   Door 1
                   Door 2
                   Door 3
                   Close all
                

                `
            })
            export class AppComponent {
              door: number;
              setDoor(num: number) {
                this.door = num;
              }
            }
