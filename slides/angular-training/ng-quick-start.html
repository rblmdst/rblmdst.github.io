Angular Quick Start

Purpose

Create a basic Angular application and run it using the Angular CLI.

Angular CLI

Angular CLI is Tool used to create and manage an Angular app. It provides the ability to:

  • create a project from scratch
  • scaffold components, directives, services, etc.
  • lint your code
  • serve the application
  • run your unit tests and end to end tests.

                # install the angular-cli globally
                $ npm install -g angular-cli

Show command help 


              # show help for all command
              ng --help

              # show help for a specific command
              # ng <cmd> --help
              ng new --help

Create a new app

There are two ways to create a new application using Angular CLI : 


              # create a new application in the current directory
              $ ng init

or 


              # create a new directory and run ng init inside the new directory
              $ ng new <app-name>

ng new is similar to ng init, except that it also creates a directory for you.

Creating a new app 


                # create a new application
                $ ng new myApp

    The command will :
  • create a new folder myApp (containing a default project structure) in the current directory;
  • install dependencies;
  • create a typescript configuration files;
  • set up the project directory (myApp) as new git repository;
  • generate Angular CLI workspace file (angular.json).

Angular project structure

After creating the project, go to the generated project folder 


              # go to the generated project folder
              cd myApp
